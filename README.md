## 🎯 Présentation
API backend de réservation de rendez-vous pour Stellaris Conseil. Gère les créneaux, les confirmations par email (SMTP), et expose une API sécurisée par clé pour les opérations admin.

---

## ✨ Fonctionnalités
- ✅ API REST avec authentification par clé API
- ✅ Gestion des créneaux de rendez-vous
- ✅ Confirmations et refus par email (SMTP)
- ✅ Interface admin

---

## 🚀 Installation
```bash
git clone https://github.com/MattiaPARRINELLO/Stellaris.git
cd Stellaris

npm start
```
